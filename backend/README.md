# Backend - Kiosco Educativo

## Controladores Instalados

Los controladores están instalados en `node_modules/`:
- `pg` - Controlador de PostgreSQL
- `ws` - WebSocket
- `express` - Servidor web
- Y otras dependencias necesarias

## Comandos

- `npm install` - Instalar controladores (ya está hecho)
- `npm start` - Iniciar servidor
- `npm run init-db` - Inicializar base de datos

## Configuración

El archivo `.env` contiene las configuraciones de PostgreSQL.
